# OpsClone Landing Page

**Clone Your Best Operator**

AI that watches how your top performer works, then replicates their decisions across your entire operation.

## Tech Stack

- **Next.js 14** (App Router, static export)
- **React 18**
- **Three.js** via @react-three/fiber + @react-three/drei
- **Tailwind CSS**
- **TypeScript**

## Features

- 3D network visualization of operational knowledge capture
- Fully responsive design
- Glassmorphism UI
- CSS-only animations (no external libraries)
- Static export ready for deployment

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Generates static export in `out/` directory.

## Deploy

Deploy the `out/` directory to any static hosting:
- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages
